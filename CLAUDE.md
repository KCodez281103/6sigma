# 6sigma - Promeera Project

## Project Overview
**6sigma** is the flagship product of Promeera, an early-stage startup building real-time market intelligence tools for GTM (Go-To-Market) leaders at high-growth B2B SaaS companies.

## Product Description
6sigma is a real-time market intelligence feed that provides competitive intelligence filtered for specific markets and connected to sales pipelines. Think: Bloomberg terminal for GTM teams, built for 2026.

## Target Audience
- GTM leaders (CROs, VP Sales, VP Marketing)
- High-growth B2B SaaS companies
- Series B and C stage companies (initial focus)

## Relationship to Promeera

## Design Philosophy
- **Not** a generic SaaS landing page
- **Not** a startup template
- **IS** a precision instrument
- Sharp, fast, intelligent
- Built to immediately resonate with CROs and GTM leaders

## Brand Identity
- **Visual**: Bloomberg terminal aesthetic, modern, 2026
- **Colors**: Dark backgrounds (#080810), electric blue (#3B82F6), no purple, no gradients
- **Typography**: IBM Plex Mono (headlines/data), DM Sans (body)
- **Tone**: Professional, precise, serious, high-signal

## Key Value Propositions
1. Real-time competitive intelligence (vs. manual tracking)
2. Market-filtered signals (vs. noise)
3. Pipeline-connected insights (vs. disconnected data)

## Competitive Advantages
- Reduces 47 hours/month of manual tracking
- Prevents $340K pipeline impact from untracked competitor moves
- Cuts response lag from 30-60 days to real-time

## Tech Stack
- Landing page: Single HTML file (vanilla JS, embedded CSS)
- No external dependencies except Google Fonts
- Fully responsive
